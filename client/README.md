# PipeFlow CRM
### Full-Stack CRM Application for High-Velocity Sales Teams

PipeFlow is a modern full-stack CRM Lead Management System built as part of a Software Engineer Internship Assessment.  
The application helps sales teams manage leads, track deal progress, monitor pipeline performance, and organize sales activity through a clean and responsive interface.

---

## 📌 Project Overview

This project was developed to demonstrate full-stack development skills including:

- Frontend development
- Backend API development
- Authentication & authorization
- Database integration
- CRUD operations
- Dashboard analytics
- Search & filtering
- UI/UX design
- Problem-solving & debugging

The system is inspired by modern CRM platforms such as HubSpot, Pipedrive, and Salesforce while maintaining its own custom design and workflow.
